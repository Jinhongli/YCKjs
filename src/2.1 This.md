# Chapter 1. This

## 为什么要用`this`

简单来说，就是实现代码复用。举个栗子：

```javascript
function say() {
    console.log(this.name + ' say ' + this.voice + '.');
}
var cat = {
    name: 'kitty',
    voice: 'meow'
};
var dog = {
    name: 'snoppy',
    voice: 'bark'
};
say.call(cat); // "kitty say meow."
say.call(dog); // "snoppy say bark."
```

同一段代码`say()`可以根据执行环境的不同而改变结果。

## 规则

首先要清楚`this`本身：

1. `this`不是代表当前的函数。
2. `this`是一个当前作用域下的变量。
3. `this`的值仅与函数的执行方式有关。
